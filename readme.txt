Языковой перевод MaxSite CMS
----------------------------

См. http://wiki.max-3000.com/pmwiki.php?n=Main.Language


Установка языка
---------------

Скопируйте языковые файлы в «application/maxsite/language». 

Язык указывается в «application/maxsite/mso_config.php». 


********************************************************************************
По всем вопросам, связанным с переводом, обращаться только через
форум http://forum.max-3000.com/ или непосредственно к автору перевода.
